# 演示数据账号

项目提供一个只在 `demo` profile 下启用的演示数据初始化器。启动后会创建或重置演示账号，并刷新一套适合实机演示的数据。

- 密码：`Demo@123456`
- 昵称：`林知远`

## 启动方式

演示数据写入当前 `application.yml` 指向的 MySQL 数据库。启动前请确认本机 MySQL 可用，默认连接信息为：

- 地址：`localhost:3306/smart_exam`
- 用户名：`root`
- 密码：`111111`

如本机 MySQL 账号不同，请先修改 `backend/src/main/resources/application.yml`。

```powershell
cd backend
mvn.cmd spring-boot:run "-Dspring-boot.run.profiles=demo"
```

初始化器会在 `demo` profile 启动时刷新 `demo_student` 的演示业务数据，避免旧数据量不足。清理范围只限该演示账号，不影响其他用户。
